NAME
Creative Corner

DESCRIPTION
This site is a class project for CISP220, Web Page Applications. It is focused on combining HTML, CSS, and JavaScript to make a user-friendly and problem-solving website. I chose to do a crafting website because I know that I would find it useful and I hope that other people would too!
This site includes features like dark mode, Google Translate, responsive window sizing, a nav bar, and clicking on images/text to go to a new page. 

INSTALLATION 
1. Download or clone the repository
2. Open the project folder in Visual Studio
3. Run index.html in a web browser 

USAGE
Once the project is opened in a browser, you can: 
- Navigate through pages using the nav bar
- Click on a project to go to the associated page to see the directions
- Change the window size to see the container shrink
- Change the site to dark mode with the button 
- Translate the pages with the Google Translate button
